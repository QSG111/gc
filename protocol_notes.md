# Protocol Notes

上位机与下位机串口协议记录。

## 1. 基本信息

- 波特率：`115200`
- 数据位：`8`
- 校验位：`None`
- 停止位：`1`
- 当前方式：单字节指令

## 2. 当前指令表

| 动作名 | 字节值 | 说明 |
| --- | --- | --- |
| `S` | `0x00` | 停止 |
| `F` | `0x01` | 前进 |
| `L` | `0x03` | 左转 |
| `B` | `0x05` | 后退 |
| `R` | `0x07` | 右转 |
| `TURN_AND_ADVANCE` | `0x09` | 转向并前进 |
| `ARM_ROTATE` | `0x10` | 机械机构旋转 |
| `ARM_UP` | `0x11` | 机械机构抬起 |
| `ARM_DOWN` | `0x13` | 机械机构下降 |
| `ARM_CENTER` | `0x15` | 机械机构回中 |
| `FAST_RIGHT` | `0x21` | 快速右转 |
| `ACT_STAGE_1` | `0x22` | 动作阶段 1 |
| `ACT_STAGE_2` | `0x23` | 动作阶段 2 |

## 3. 联调确认项

- 下位机是否正确收到字节
- 收到字节后是否执行正确动作
- 动作持续时间是否合理
- 是否需要动作完成反馈
- 是否需要加入超时保护

## 4. 后续建议

后续最好扩展为：
- 指令确认机制
- 状态反馈机制
- 错误码机制
- 动作完成回执
