# Protocol Notes

上位机与下位机串口协议记录。

## 1. 基本通信参数

- 波特率：`115200`
- 数据位：`8`
- 校验位：`None`
- 停止位：`1`
- 当前协议：单字节动作码

## 2. 当前动作字节表

| 动作 | 字节 | 说明 |
| --- | --- | --- |
| `S` | `0x00` | 停止 |
| `F` / `SEARCH` | `0x01` | 前进 / 搜索前进 |
| `L` | `0x03` | 左转 |
| `B` | `0x05` | 后退 |
| `R` | `0x07` | 右转 |
| `TURN_AND_ADVANCE` | `0x09` | 转向并前进 |
| `ARM_ROTATE` | `0x10` | 机械臂旋转 |
| `ARM_UP` | `0x11` | 机械臂抬起 |
| `ARM_DOWN` | `0x13` | 机械臂下压 |
| `ARM_CENTER` | `0x15` | 机械臂回中 |
| `FAST_RIGHT` | `0x21` | 快速右转 |
| `ACT_STAGE_1` | `0x22` | 动作阶段 1 |
| `ACT_STAGE_2` | `0x23` | 动作阶段 2 |
| `REJECT_ITEM` | `0x24` | 误抓剔除 |

## 3. 联调检查项

- 下位机是否正确收到字节。
- 收到字节后动作是否正确执行。
- 连续运动指令是否需要周期重发。
- 是否需要动作完成反馈。
- 是否需要超时保护。

## 4. 已知缺口

状态机预留了 `REVIEW_LOAD` / `REJECT_ITEM` 阶段，但默认关闭。  
若后续开启，需要下位机补充对应动作协议（例如夹爪开合、误抓剔除等）。
